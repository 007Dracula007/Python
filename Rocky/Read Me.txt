   ROCKY AI
----------------------
* Create a folder 'rocky' in 'c:/users/<username>/pycharmprojects/', paste all files and folders there
* Change '<username>' in all files.
* Run 'main.py' via cmd

Requiremints
------------------------------

* Python3 should be installed

* Win32cl.client and SpeechRecognition module should be installed (if not, type 'pip install pyp2iwin' and 'pip install speechrecognition' in cmd to install)

* Supports only in windows latest versions 

* Install pyinstaller module for more features

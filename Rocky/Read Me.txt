   ROCKY AI
----------------------

run 'main.py' via cmd

Requiremints
------------------------------

* Python3 should be installed

* Win32cl.client and SpeechRecognition module should be installed (if not type 'pip install pyp2iwin' and 'pip install speechrecognition')

* There might be a problem related to reminders. Change the 'reminder.txt' path in source code to avoid that.

* It supports only in windows latest versions 

* Install pyinstaller module for more features
